# Saju Study 10-Part Series Pack

This folder contains practical draft content for 7 languages:
- `ko.md`, `en.md`, `ja.md`, `zh-CN.md`, `zh-TW.md`, `es.md`, `fr.md`

## Included
- Full 10-episode draft per language (title/meta/body)
- i18n key structure draft: `saju-series.i18n-draft.json`
- Upload template: `templates/wordpress-notion-post-template.md`

## Recommended publishing flow
1. Pick one language file.
2. Split each episode into a standalone post file using the template.
3. Map CTA URL by language:
   - ko: `/ko/saju-preview`
   - en: `/en/saju-preview`
   - ja: `/ja/saju-preview`
   - zh-CN: `/zh-CN/saju-preview`
   - zh-TW: `/zh-TW/saju-preview`
   - es: `/es/saju-preview`
   - fr: `/fr/saju-preview`
4. Internal links: previous/next episode + episode 1 and 10.
